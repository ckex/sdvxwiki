### Specs
Newest generation valkyrie cabs use Bemani PC Type 12 (GELDJ-xx).
<div className="flex items-center justify-center mt-5">
  |Type 12 Bemani PC| Part |
  |:-:|:-:|
  |**Motherboard**| ASUS H310-IM-A-KAN |
  |**CPU**| Intel Core i5-9500E |
  |**GPU**| GIGABYTE GeForce GTX 1650 1530MHz 4GB |
  |**RAM** |16 GB DDR4-2133 8GB x 1 Single Channel |
  |**OS**| Windows 10 |
  | **Monitor** | 43" 4k 120hz monitor\* |
</div>

\*Even though it has the capability to run 4k, voltex downscales to 1920 x 1080

- Older Valkyrie cabs might have a Supermicro C7B360-CB-M motherboard.
- Older Valkyrie cabs might have a Intel Core i5-9400F 2.9GHz CPU
